# Vue 3 + TypeScript + Vite

1. 简历模版
  新增模版需要按照 `template1` 新建文件夹，然后在 `template1` 文件夹中新建 `index.vue` 文件,以此类推 `template2`, `template3` 等。

2. 模板切换
   项目切换模版需要在 `App.vue` 修改 `template` 属性为 `template1`, `template2`, `template3` 等。

3. 新增组件模块
   需要新增组件模块在 `components` 文件夹中新建页面，新增新的页面组件模块

# FAQ
