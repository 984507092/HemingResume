## 云小派(微信小程序/支付宝小程序)

## 项目描述

云小派是为了孕妈和宝妈用户人群而开发的小程序，作为前端开发工程师，负责整个项目的搭建和后续的开发/维护，以及对公共 组件模块的封装

## 项目职责

从 0 到 1 完成了小程序的开发，实现用户注册登录，无感刷新，个人信息等核心功能
对小程序进行性能优化，页面平均响应时间由2s 降低至500ms,提高了小程序流畅度和稳定性
主要负责封装组件比如 popup、模态框、手风琴、分享海报、到计时、input 实现双向绑定等组件
负责小程序的申请和备案工作，配置小程序的各种环境变量，例如AppID、密钥等，成功将小程序发布上线;
难点：对登录实现 token 过期无感静默登录功能和用户、平台、机构三方角色校验
难点：使用原生 CANVAS 封装分享海报和二维码功能，支付宝和微信同时兼任安卓和IOS 的问题

### 项目业绩

对登录实现 token 过期无感静默登录功能和用户、平台、机构三方角色校验
使用原生 CANVAS 封装分享海报和二维码功能，支付宝和微信同时兼任安卓和IOS 的问题

## 宠采采后台管理系统

### 项目描述

本项目是一个关于宠物信息的 pc 的后台管理系统，以及不同账号的权限以及动态路由的配置。

### 项目职责

1. 主要负责静态页面的布局和前后端的交互
2. 使用插件粒子效果使页面效果看起来更加的美观。
3. 使用 axios 对 request 和 response 的请求拦截的配置，
4. 使用 beforeEach 的拦截，对用户要去的隐私页面的拦截，路由使用 hash 模式配置动态路由与懒加载，5.负责登录页面的维护以及 vue-Router、pinia 的状态管理 6.封装路由信息 , 使用addRoute实现动态路由权限；

## 宠采采小程序

### 项目描述

本项目是一个基于 uniapp 开发的项目，主要是一些关于宠物的商品，为用户提供一个方便简捷的 App。

### 项目职责

1.通过蓝湖根据 UI 高度还原项目视图，项目中使用 rpx 来做移动端适配使用
normalize.css 来抹平各浏览器之间样式差异，编写样式使用 SCSS 预编译，3.在搜索页面对输入框进行函数防抖可以节省性能 4.负责购物车全选、反选、单选、计算总价、删除等功能 5.对省市区的 json 数据进行封装，为了在多个地方使用时方便 6.对 axios 进行二次封装，实现状态码的分离处理，达到给用户友好提示的效果 7.使用 vuex 的 module 实现数据的模块化管理，形成一个 namespaced 的划分；8.负责订单状态，取消订单，删除订单以及发货物流的信息的端口对接

## 研美管理

### 项目描述

一个健身房，美容院的活动配置系统，对运营和门店的活动进行管理，配置不同门店，不同城市生成不同的活动，并进行数据统计。

### 项目职责

通过 vue-cli ，结合eslint + prettier + husky + lint-staged + commitlint 搭建现代化前端工程，通过 css 变量配置一键换肤；
二次封装axios， 通过配置请求，响应拦截器，统一处理接口返回结果，添加全局的 loading，message；
通过 EChars 实现数据可视化以及地图部分的需求；
通过 vite-press 搭建运营帮助文档；
通过 websocket 实时展示现场的告警信息；
通过 ElementUI 实现可编辑表格，单元格合并；
使用 vue-seamless-scroll 插件解决自动滚动的问题
通过 Vue-Draggable 实现表单设计器的拖拽功能；
通过 Axure 完成原型设计，并参与原型评审。
